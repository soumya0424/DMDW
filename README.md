# DMDW

Data Mining and Data Warehousing lab projects and exercises.

## Overview

This repository contains Jupyter Notebook projects for the DMDW (Data Mining & Data Warehousing) course. It covers various data mining techniques applied on real-world datasets.

## Tech Stack

- Python
- Jupyter Notebook
- Pandas, NumPy
- scikit-learn
- Matplotlib / Seaborn

## Datasets Used

- Credit Card Fraud Detection
- Diabetes
- Heart Disease
- IMDb Movies India
- IRIS
- Titanic
- Uber NYC Trips
- Wine Quality
- Sales Prediction

## Getting Started

```bash
pip install jupyter pandas numpy scikit-learn matplotlib seaborn
jupyter notebook
```
