## Overview

DMDW is a collection of Data Mining and Data Warehousing lab notebooks applying various mining techniques to real-world datasets.

**Why DMDW?**

- 🔍 **Dataset Variety:** Credit card fraud, diabetes, heart disease, IRIS, Titanic, Uber NYC, and more.
- 📊 **Mining Techniques:** Classification, clustering, regression, and pattern mining.
- 🤖 **ML Integration:** Uses scikit-learn for model building and evaluation.
- 📈 **Visual Analysis:** Matplotlib and Seaborn plots for data exploration.
- 🧪 **Hands-On Labs:** Progressive complexity across multiple lab assignments.

## Getting Started

### Prerequisites

This project requires the following dependencies:

- **Programming Language:** Python 3.8+
- **Packages:** `scikit-learn`, `pandas`, `numpy`, `matplotlib`, `seaborn`, `jupyter`

### Installation

1. **Clone the repository:**

    ```bash
    git clone https://github.com/soumya0424/DMDW
    cd DMDW
    ```

2. **Install dependencies:**

    ```bash
    pip install scikit-learn pandas numpy matplotlib seaborn jupyter
    ```

### Usage

```bash
jupyter notebook
```
